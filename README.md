# Task 6 - SQL Sales Trend Analysis

## 📌 Project Overview
This project analyzes online sales data using SQL in MySQL Workbench. The goal is to calculate monthly revenue and order volume, identify top-performing months, and practice SQL aggregation and date functions.

## 🎯 Objective
- Analyze monthly sales performance.
- Calculate total revenue.
- Count order volume.
- Identify the top 3 months by revenue.
- Practice SQL queries using aggregate and date functions.

## 🛠️ Tools Used
- MySQL Workbench
- SQL
- CSV Dataset (Online Sales)


## 📈 Analysis Performed
1. Imported the CSV dataset into MySQL Workbench.
2. Created the `sales_db` database.
3. Calculated monthly revenue.
4. Calculated monthly order volume.
5. Identified the top 3 months based on revenue.
6. Displayed and analyzed the query results.

## ✅ Results
- Successfully imported the online sales dataset into MySQL.
- Calculated monthly revenue using SQL aggregate functions.
- Counted monthly order volume.
- Identified the top-performing months based on revenue.

## 📚 Learning Outcomes
- Database creation and data import.
- SQL aggregation and grouping.
- Date and time functions in SQL.
- Sales trend analysis using SQL.
- Organizing a professional GitHub project.
